The Work

"Every system tells a story. These are mine."

Live — Firebase Deployed
Cascade AI — Multi-Agent Intelligence Platform
Python  /  OpenAI API  /  LangChain  /  Firebase Cloud Functions  /  Firestore  /  GCS  /  Node.js

A production-deployed backend where an LLM planner decomposes user tasks into a coordinated pipeline of specialised agents — web search, scraper, auditor, router, and formatter — executed in sequence via Firebase Cloud Functions. Not a demo. Not an API wrapper. A real orchestration system with self-healing branching logic and private per-user memory.

  • LLM planner generates multi-step agent blueprints from plain text
  • Parallel search pipeline with AI-powered auditing of scraped pages
  • Router agents with conditional branching and automatic index correction
  • Per-user context in Firestore; large payloads offloaded to GCS
  • Output formatters for Slack, PDF, and email
  • Firebase Auth, onboarding quiz, private memory layer per user
AWS EC2 — Live Demo
OptiQuant — Full-Stack ML Application
Python  /  LightGBM  /  CatBoost  /  Random Forest  /  SHAP  /  Streamlit  /  Docker  /  AWS EC2  /  GitHub Actions

An end-to-end ML application from feature engineering and ensemble model training to a live Streamlit interface for generating and explaining stock alpha signals. Walk-forward backtesting prevents data leakage. SHAP explainability makes every prediction inspectable in the UI. Docker and CI/CD make every model update seamless.

  • Weighted ensemble of LightGBM, CatBoost and Random Forest
  • Walk-forward validation simulating real production retraining
  • Dockerised and deployed to AWS EC2 via GitHub Actions CI/CD
  • SHAP attribution layer surfaced directly in the Streamlit UI
  • 161% returns  |  2.73 Sharpe  |  64% win rate over 272 days
  • Live prediction mode and backtesting engine in one interface
